# US004 - Contact Form Validation

## Overview
Verify empty submission and invalid email validation on Contact form.

## Steps
- Open Contact page
- Submit empty form and validate errors
- Fill invalid email and validate email error
- Confirm success message is not shown for invalid submission

## Assertions
- Form errors are visible
